Укажите любое количество адресов в левое поле, а в правое необходиое название для адресов на Bybit. Например если в поле Label ввести "Wallet-", то будут добавляться адреса с названием "Wallet-i", где i - порядковый номер кошелька. Запустите код на https://www.bybit.com/user/assets/money-address/batch. Будут созданы функции add() (увидите при запуске в консоли), которые будут поочередно добавлять адреса. Для запуска этих функций введите add1(), add2() и т.д.
